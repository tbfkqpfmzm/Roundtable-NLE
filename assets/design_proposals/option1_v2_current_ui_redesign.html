What Changed (vs. your current UI)

⬅ CHARACTER FILTER (was: tall thumbnail grid)

  • Instead of tall 90×90 thumbnails → compact chip list (Photoshop layers style)
  • Character count badges — see at a glance how many shots use each character
  • All / Unassigned as chips — same function, more compact
  • Inline search — filter characters by name right in the filter panel
  • Wider column — 200px instead of ~106px, fits character names better

⬅ SHOTS COLUMN (was: flat list + combo)

  • Auto-grouped into sections — ⭐ Favorites, 📁 Character groups, Recent
  • Sort dropdown — Sort by A–Z, Recent, Character, Favorites
  • Star/favorites toggle — ⭐ pin shots to top (same as Photoshop's star in layers)
  • Character & BG tag badges — see what's in each shot without opening it
  • Removed the redundant combo dropdown — the sorted list + groups replaces it
  • View toggle buttons — switch between list and grid views (stub for future)

✅ Everything else — the rail, the preview area, the library tabs, the layers panel, the properties panel — stays exactly as you have it now. Only the two left columns received the Option 1 treatment.
