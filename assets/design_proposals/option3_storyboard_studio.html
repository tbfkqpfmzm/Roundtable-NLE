Key Improvements

  • Storyboard canvas — Shots are displayed as visual cards on a free-form board, not a linear list. Users can arrange, group, and sequence shots visually — like a real storyboard.
  • Horizontal character strip — Instead of a tall narrow column, character filters live in a compact horizontal strip above the board. Much more screen-efficient.
  • Scene grouping — Shots are automatically grouped into scenes with labeled dividers. Users can drag shots between scenes or create new ones.
  • Card-based shot thumbnails — Each shot is a card showing duration, characters used, layer count, and FPS at a glance. Hover reveals quick-action buttons (duplicate, delete).
  • Timeline ruler — The left edge shows shot sequence numbers, helping users think about the order of shots in their video project.
  • Unified inspector panel — Properties, layers, tags, and library all live in a single tabbed inspector. Less panel switching, more context.
  • Dot-grid canvas background — The board has a subtle grid pattern that makes it feel like an actual creative workspace (similar to Figma/Miro boards).
  • Breadcrumb navigation — Shows Characters / Character Name / Shot Name hierarchy. Click any crumb to navigate up.
  • Workspace selector — Users can have multiple scene workspaces, keeping shots organized by narrative structure, not just by character.
  • Tag cloud — Backgrounds and tags are displayed as clickable pills in the inspector. Add custom tags to organize shots beyond just character names.
  • View toggle — Switch between Board, List, and Grid views to match your workflow preference.
  • Add shot placeholder — A dashed "+" card at the end of the board makes it obvious how to create new shots.
