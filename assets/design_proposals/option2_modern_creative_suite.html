Key Improvements

  • Slim icon rail — Minimal 56px rail frees up space for content. Icons-only navigation keeps the UI uncluttered.
  • Pill-based character filter — Horizontal scrollable pills with character avatars & shot counts sit at the top of the panel. Much more compact than the current thumbnail grid.
  • Split-panel browsing — Shots (top) and character avatars (bottom) live in one unified panel with a draggable splitter. Shots list shows layers count, character tag, and recency.
  • Unified search — A single search bar at the top searches both shot names AND character names simultaneously.
  • Circular character avatars — Filter by character uses round avatar tiles instead of square thumbnail buttons, making it visually distinct from shots.
  • Right context panel — Quick-access icons for layers, properties, history, and info. Can be collapsed entirely for more content space.
  • Status badge — "Saved" indicator gives confidence that your work is persisted without needing to check.
  • Drag-handle on split panel — User can resize the shots vs characters split to prioritize what they need most.
