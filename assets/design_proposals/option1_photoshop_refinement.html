Key Improvements

  • Layers-panel-style character filter — Compact chip list with character counts instead of a tall thumbnail grid. Familiar to Photoshop users.
  • Auto-grouped shots — Favorites, character folders, and recents are visually separated. Shots can be dragged between groups.
  • Star/favorites system — Mark any shot as a favorite to pin it to the "⭐ FAVORITES" group at the top.
  • Sort dropdown + search — Sort by name, date, character, or last used. Search filters across both shot names and character names in results.
  • Tag badges on shots — Each shot shows which characters/backgrounds it uses at a glance without opening it.
  • Shot count badges — Each character filter chip shows how many shots use that character.
  • Breadcrumb navigation — Shows COMPOSE › Shot Name context in the content header.
